# DefconZ - Sitio estático apocalíptico

Landing inspirada en temática apocalipsis zombie para tu servidor. Incluye secciones de servidor, modalidades, Discord e Instagram con placeholders listos para reemplazar.

## Estructura
- index.html: página principal.
- styles.css: estilos oscuros con acentos rojos y efecto glitch.
- main.js: botón para copiar la IP del servidor.

## Cómo usar
1) Abre index.html en tu navegador para previsualizar.
2) Edita index.html y reemplaza los placeholders:
   - IP: play.defconz.tld (sección Servidor y botón copiar).
   - Nombre: DEFCON-Z.
   - Enlaces Discord/Instagram en los botones y chips.
   - Textos de modalidades y estado.
3) Ajusta la paleta o tipografía en styles.css si lo deseas.

## Publicar en GitHub Pages
- Sube estos archivos al repositorio.
- En GitHub, ve a Settings → Pages → Source: deploy from branch → main (root). Guarda.
- Espera el deploy y visita https://TU-USUARIO.github.io/TU-REPO/.

## Personalización rápida
- Cambia colores en styles.css en la sección :root.
- Ajusta el glitch editando la clase .glitch.
- Agrega más tarjetas de modalidades duplicando un article en la grilla.

## Requerimientos
- Sitio estático; no necesita build ni dependencias.
